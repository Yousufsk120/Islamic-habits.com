# 100 Duas Multilingual Publication Plan

## Recommended Language Set

The publication can cover six audiences by keeping the Arabic source and adding five translations:

- Arabic original
- English
- Bengali
- Japanese
- Simplified Chinese
- Malay, Bahasa Melayu

Optional later editions:

- Indonesian
- Urdu / Hindi
- Korean
- French
- Spanish

## Publication Rule

Do not publish machine translation as final religious text. Use this workflow:

1. Extract every dua into a source table.
2. Preserve Arabic exactly.
3. Preserve reference exactly.
4. Add transliteration only where useful.
5. Translate meaning, not word-by-word literal output.
6. Review by a qualified Islamic reviewer.
7. Review each language by a native speaker.
8. Publish with a clear note: translations are approximate meanings.

## Source Table Fields

Each dua should become one structured record:

- ID
- Category
- Arabic
- Transliteration
- English meaning
- Bengali meaning
- Japanese meaning
- Simplified Chinese meaning
- Malay meaning
- Source reference
- When to recite
- Benefit / lesson
- Review status

## Suggested Output Formats

- Web app dua library
- Downloadable PDF per language
- Combined multilingual PDF
- Social carousel set
- Audio-ready script
- SEO pages such as `/duas/morning`, `/duas/quranic`, `/duas/forgiveness`

## Website Integration

Add a `Dua Library` section to Islamic Habits with:

- Search by need: forgiveness, protection, anxiety, family, rizq, akhirah
- Filter by source: Qur'an, authentic hadith
- Language selector
- Copy/share button
- Save favorite duas
- Audio placeholder for future recitation
- Reference visible on every dua card

## Manual Interventions Needed

- Confirm copyright/permission for the PDF if it is not your original work.
- Choose whether the sixth language is Arabic original or another translated language.
- Provide official contact email for the website.
- Arrange scholar review for Arabic, reference, and religious wording.
- Arrange native review for Japanese, Chinese, Bengali, and Malay.
- Decide brand name on the PDF cover: Islamic Habits or Islamic Habits.

## Translation Notes

- Japanese: use respectful, accessible wording for beginners. Avoid overly academic phrasing.
- Chinese: use Simplified Chinese first. Traditional Chinese can be a later edition.
- Bengali: preserve familiar Islamic terms where natural, such as dua, tawbah, akhirah.
- Malay: use Bahasa Melayu. Avoid calling it “Malaysian” in publication metadata.
- English: keep simple, global English for new Muslims and non-native speakers.

## Quality Checklist

- Arabic text verified.
- References verified.
- No weak/unauthenticated claim is presented as authentic.
- Translation marked as meaning, not Qur'an replacement.
- Layout supports right-to-left Arabic and left-to-right translations.
- No ads placed between Arabic dua text and its reference in a disrespectful way.
